# Internationalization (i18n) Guide

## Supported Languages

Currently, the Image Details extension supports the following languages:

- **English (en)** - Default language
- **Portuguese - Brazil (pt-br)** - Brazilian Portuguese

## How It Works

The extension automatically detects the VS Code language setting (`vscode.env.language`) and displays all UI text in the appropriate language.

## Language Detection Logic

1. First, checks for an exact match (e.g., `pt-br`)
2. If no exact match, checks for the base language (e.g., `pt` from `pt-BR`)
3. Special handling for Portuguese variants (always defaults to `pt-br`)
4. Falls back to English if no match is found

## Adding a New Language

To add support for a new language:

1. Open `src/imageDetailsEditor.ts`
2. Find the `translations` object (around line 10)
3. Add a new entry with your language code following this template:

```typescript

    imageDetails: 'Translation',
```

## Testing Your Translation

1. Change your VS Code language setting:
   - Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
   - Type "Configure Display Language"
   - Select your language
   - Restart VS Code

2. Open an image file with the extension
3. Verify all labels are correctly translated
